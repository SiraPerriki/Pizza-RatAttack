## PIZZAS y RATAS (Godot)

Minijuego vertical: lanzas una pizza con tirachinas, rebotas en una “goma” inferior y cazas ingredientes que pasan por la parte superior. Evita las ratas: muerden la pizza y quitan vidas.

### Requisitos

- Godot 4.x

### Cómo abrir

1. Abre Godot.
2. Importa el proyecto seleccionando esta carpeta.
3. Ejecuta la escena principal `scenes/Main.tscn`.
